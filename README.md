### Commands

- eb!add `Name` `MM-dd-yyyy`
- eb!remove `Name`
- eb!reminderChannel `channelName`

You will recieve a reminder message on the `reminderChannel` like:

Today is `Name`'s birthday! :tada: :tada:
(followed by a related image)
